3D-Placer User Manual

Navigation

Use the sidebar menu to navigate between different sections of the app.

Model Library

This is where all your 3D models are stored. You can:

  • View all imported 3D models
  • Import new models from your device
  • Delete unwanted models
  • Select models to edit or place in scenes

Model Editor

Customize individual 3D models with the following tools:

  • Apply textures to models
  • Adjust texture mapping (Box or Cylinder projection)
  • Modify texture scale and offset
  • Generate new thumbnails
  • Load textures from your gallery

Scenes

Manage your 3D scenes:

  • Create new empty scenes
  • View all your saved scenes
  • Rename scenes as needed
  • Delete unwanted scenes

Scene Editor

Build and customize 3D environments:

  • Add models from your library
  • Position, rotate, and scale objects
  • Change lighting conditions (High Noon, Twilight, Night)
  • Save your scene arrangements

Settings

Customize app behavior:

  • Toggle dark mode on/off
  • Adjust grid size in 3D views
  • Reset to default settings

3D View Controls

Navigate in 3D views using these touch gestures:

Orbit (Rotate Camera)

Touch and drag with one finger to orbit around the center point.

Pan (Move Camera)

Touch and drag with two fingers to pan the view.

Zoom

Pinch with two fingers to zoom in and out, or use a two-finger scroll gesture.

Reset View

Use the "View Reset" button to restore the default camera position.

Object Manipulation

In the Scene Editor:

  • Tap on an object to select it
  • Use the transform controls to adjust position, rotation, and scale
  • Tap the background to deselect

Tips & Tricks

  • For better performance, use models with lower polygon counts
  • Save your work frequently
  • Use lighting presets to showcase your models differently
  • Texture resolutions of 512px or 1024px offer good quality without excessive memory usage
